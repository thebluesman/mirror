# History

Archive of superseded and completed planning docs — pre-spike scoping, the
three-spike R&D arc's planning docs, and the shipped v1/v2 PRDs and build
plans. Kept for provenance ("how did we get here"), not as a reading list for
current state — start at the root `CLAUDE.md` for that.

**Frozen, like `spike/`:** these files are not edited after archiving. A
correction to a decision recorded here goes in `docs/adr/` or the live doc it
now belongs to, not back into these files.
